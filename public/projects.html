Honing JavaScript via Exercism: JavaScript Exercises 2013

As a tool to learn how to better code in JavaScript (JS), I joined Exercism.io and started practicing writing JS while solving simple string manipulation exercises. Not only did I make all the provided tests pass, but then I submitted my solutions to the community and refactored those solutions until the community was pleased with their readability and simplicity.

While waiting on feedback from the community regarding my code, I also spent time critiquing JS solutions submitted by others. In some ways, critiquing others' code felt more valuable, as it often led to researching best practices in JS before I'd give feedback.

JavaScript Generated Calendar: 2010 (Refactored 2013)

The calendar you see below is automatically generated by JavaScript. Fresh out of school and curious about JavaScript, I found some broken code on the web where the author stated "if you can fix this, you can have it." I fixed it and then added a bit to it:

  • Fixed the layout itself so that the correct date would be displayed under the correct day of the week.
  • Added code to make sure that every month of the year displayed correctly--including leap years.
  • Added code to show today's date in bold-face.
  • As noted above, when I resurrected this code while starting this website, I refactored it to be more readable, but it still could use some work.
